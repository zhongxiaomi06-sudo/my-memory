**匹配评分流程**：
```
用户输入JD → /api/fast-match (POST JSON)
  → _extract_skills() 提取技能关键词
  → _search_candidates() SQLite全文搜索
  → HybridEngine.compute(candidate, job)
    → RuleEngine (5规则: skill 0.35 + exp 0.2 + company 0.15 + salary 0.1 + stability 0.1)
    → LightweightPredictor (ML模型权重)
    → blended = 0.6 × rule + 0.4 × ml
  → _generate_explanation() 中文自然语言解释
  → _save_matches() 存入 matches 表
  → 返回 { matches, total, job_id }
```
