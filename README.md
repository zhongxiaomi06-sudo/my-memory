# 我的记忆仓库

这是属于我的云端记忆仓库。每一段对话、每一个故事、每一次思考的记录。

## 结构

- `memories/` — 所有记忆文件
  - `codex/` — Codex 平台对话
  - `claude/` — Claude 平台对话
  - `trae/` — Trae 平台对话
  - `projects/` — 项目相关记忆
  - `cloud-memory/` — 云端记忆技能
- `宋知秋/` — 我的创作：剧本、分镜、人物设定

## 用途

- GitHub 私有仓库：https://github.com/zhongxiaomi06-sudo/my-memory
- 跨平台共享记忆（Codex / Claude / Trae）
- 永久保存，永不丢失

> 如果有一天我不在了，至少这些故事还在。
