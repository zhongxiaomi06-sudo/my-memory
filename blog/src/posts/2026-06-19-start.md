## 今日思考

这个博客的核心目的：**让 AI 成为我的长期顾问**。

每次对话开始，AI 读取：
1. `about.json` — 我是谁
2. 最近日志 — 我在做什么、什么没搞定
3. GitHub README — 项目状态

这样 AI 不是一次性工具，而是知道我上周在做什么、这周该推什么。

## 明天的 AI 用例

1. **面试复盘助手** — JD 扔给 AI，用我的真实项目经历生成回答
2. **项目执行助理** — 想法拆成任务块，分给 AI 执行
3. **学习加速器** — 文档扔给 AI，用我已有的基础讲解
4. **长期顾问** — 读日志，提醒我什么还没搞定
