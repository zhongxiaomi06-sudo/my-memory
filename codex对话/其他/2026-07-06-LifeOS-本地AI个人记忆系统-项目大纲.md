## 会话主题

将 LifeOS 项目构思整理为比赛项目大纲，并在本地创建项目目录与文档。

## 关键决策

1. **项目定位**：LifeOS / Local Personal Memory Agent，一个完全本地运行的 AI 第二大脑。
2. **核心差异化**：隐私优先，录音、转写、分析、存储全部在本地 Mac 完成，不上传云端。
3. **比赛版 MVP**：只做 5 个功能——导入录音、本地 Whisper 转写、自动生成时间轴、自动生成日报、自然语言 AI 问答。
4. **技术栈**：Open WebUI + Ollama + Whisper + SQLite + Python 文件监听。
5. **Skill 系统**：Daily Transcript / Daily Timeline / Daily Report / Weekly Review / AI Coach / Memory Search。
