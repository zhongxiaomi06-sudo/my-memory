## 学到什么
- `fd`/`rg`/`fzf`/`zoxide` 是现代终端核心工具
- `repo list` / `fj` / `rcd` 可以大幅加速仓库跳转
- GitHub 工作流标准路径：`branch → commit → push → PR → merge`
- `gh` CLI 可以完成 95% 的 GitHub 操作
- API key 应优先走 1Password CLI，fallback 仅作过渡
