# 科大 OpenVPN macOS 部署指南

✅ 本方案使用 macOS 系统原生 Network 设置，**无需安装任何第三方客户端**。

---

## 📥 准备工作
1. 确保以下三个文件位于本目录：
   - `ustc-dianxin.ovpn`（电信线路）
   - `ustc-cmcc.ovpn`（移动线路）
   - `ca.crt`（根证书）
2. 你的 macOS 已登录 Apple ID（确保网络权限正常）

---

## ⚙️ 添加 VPN（电信线路）
1. 打开 **系统设置 → 网络**
2. 点右下角 `+` → 类型选 **VPN** → 接口选 **OpenVPN** → 点“创建”
3. 在新设置页：
   - “配置” → 点右侧 `⋯` → **导入配置文件** → 选择 `ustc-dianxin.ovpn`
   - 系统会自动关联同目录的 `ca.crt`
4. 点右上角 ✔️ 保存
5. 返回网络列表 → 点击该 VPN 旁的 ⚙️ → **连接**

> 🔐 连接时将弹出窗口，输入 **统一身份认证账号（学号/工号）和密码**

---

## 🔄 切换线路（移动线路）
重复以上步骤，导入 `ustc-cmcc.ovpn` 即可。两个 VPN 可共存，按需切换。

---

## ✅ 验证是否成功
连接后执行：
```bash
curl -s https://ip.ustc.edu.cn
```
✅ 返回 `202.38.64.x` 或 `202.38.65.x` IP 即为科大出口，成功！

---

> 💡 提示：若导入失败，请检查 `.ovpn` 文件是否含 DOS 换行符（本文件已确认合规）
