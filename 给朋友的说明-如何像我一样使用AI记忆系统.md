# 🧠 如何像我一样使用 AI 记忆系统

> 给朋友的完整指南 — 你直接把这份文件发给 AI，它就知道怎么做了

---

## 一、先建你的记忆仓库（只需要一次）

```bash
# 1. 注册 GitHub（如果还没有）
# 打开 https://github.com 注册一个账号

# 2. 安装 GitHub CLI
brew install gh  # macOS
# 或者: https://cli.github.com/

# 3. 登录
gh auth login

# 4. 创建私有仓库
cd ~/Documents
mkdir 我的记忆
cd 我的记忆
git init
gh repo create my-memory --private --description "我的永久记忆仓库"
git remote add origin https://github.com/你的用户名/my-memory.git

# 5. 创建目录结构
mkdir -p 对话记录 项目笔记 技能文件 系统配置

# 6. 上传你的第一个文件
echo "# 我的记忆从这里开始" > README.md
git add .
git commit -m "🎉 初始化记忆仓库"
git push -u origin main
```

---

## 二、每次和 AI 对话后，保存记忆

把下面这段话发给 AI，它就会帮你保存：

---

**给 AI 的指令（复制这段发给 AI）：**

```
请帮我做以下事情：

## 1. 读取我的记忆
先检查我的记忆仓库：
- GitHub: https://github.com/我的用户名/my-memory
- 如果电脑本地有 ~/Documents/我的记忆/ 目录，直接读里面的文件
- 读取 README.md 了解我的背景
- 读取最近的文件了解我最近在做什么

## 2. 完成任务后保存记忆
每次完成一个任务或一段对话后，执行：

cd ~/Documents/我的记忆
cat > 对话记录/$(date +%Y-%m-%d)-摘要.md << 'EOF'
# 对话记录 - 202X-XX-XX

## 做了什么
（用3-5句话总结）

## 解决了什么问题
（列出关键点）

## 状态
- [x] 已完成
- [ ] 待继续

## 学到了什么
（新的知识点、技巧、灵感）
EOF

git add .
git commit -m "添加 $(date +%Y-%m-%d) 对话记录"
git push

## 3. 帮我整理归类
- 项目相关的 → 项目笔记/
- 学到的知识 → 技能文件/
- 系统配置 → 系统配置/
```

---

## 三、在新电脑/新对话中恢复所有记忆

把这段发给新的 AI：

```
请读我的记忆：
1. 如果 ~/Documents/我的记忆/ 存在，直接读取里面的所有 .md 文件
2. 如果不存在，执行：
   cd ~/Documents
   git clone https://github.com/我的用户名/my-memory.git 我的记忆
3. 读取 README.md 了解我的背景
4. 读取对话记录/ 了解最近发生的事
5. 用这些信息来理解我是谁、我在做什么
```

## 四、给你的 AI 一个"人设文件"（可选）

在仓库里建一个 `我是谁.md`，写入你的个人信息，AI 每次先读这个：

```markdown
# 我是谁

## 基本信息
- 称呼：_______
- 职业/身份：_______
- 擅长领域：_______
- 正在做的事：_______

## 我喜欢的方式
- 说话风格：_______
- 代码习惯：_______
- 需要 AI 主动做的：_______

## 我的项目
- 项目1：_______
- 项目2：_______

## 我的目标
- 短期：_______
- 长期：_______
```

## 五、完整工作流（一张图）

```
每次对话开始
  │
  ├─ AI 读取本地 ~/Documents/我的记忆/
  │   ├─ README.md        → 了解整体
  │   ├─ 我是谁.md         → 了解你的背景
  │   └─ 对话记录/最近的文件 → 了解上下文
  │
  ├─ 你正常和 AI 对话、工作
  │
  └─ 对话结束后
      ├─ AI 自动保存摘要到 对话记录/
      ├─ AI 分类整理到对应目录
      ├─ git commit
      └─ git push 到 GitHub
```

## 六、零成本、永久保存

| 服务 | 费用 | 可靠性 |
|------|------|--------|
| GitHub 私有仓库 | 免费 | 99.99%+ |
| 本地文件 | 0 | 你电脑上随时可读 |
| 不需要服务器 | 省每月几十块 | 永远不会崩溃 |

> **核心原则：不依赖任何服务器，只依赖 Git。GitHub 倒了，你本地还有；电脑丢了，GitHub 上还有。**
